# HR Analytics Dashboard 📊

An interactive HR Analytics Dashboard built using **Power BI** to analyze employee attrition, workforce trends, and department-wise insights.

## 📌 Project Overview

This dashboard helps HR teams understand:
- Employee attrition patterns
- Department-wise employee distribution
- Salary slab impact on attrition
- Gender and age group analysis
- Job role & job satisfaction correlation

## 🛠️ Tools Used

- **Power BI Desktop** – Dashboard creation & visualization
- **Microsoft Excel / CSV** – Data source (HR_Analytics-4.csv)

## 📁 Files in this Repository

| File | Description |

| `HR-Analytics Dashboard.pbix` | Power BI project file |

## 📊 Key Insights

- Overall attrition rate varies significantly across age groups
- **18-25 age group** has the highest attrition rate of **37.61%**
- **Sales department** has the most attrition among job roles
- Employees earning **6-10 LPA** show the highest attrition count
- **Female employees** have slightly higher attrition than males
